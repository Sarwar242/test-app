### 1. **Employee Management (CRUD Operations)**

- **Confirmation Dialog**: Safety measure before deletion
- **Immediate UI Update**: Optimistic UI updates

#### ‚úÖ Criteria-Based Search
- **Search Fields**:

  - Gender (dropdown filter)

- **Search State Management**: Visual indicators for active search

#### ‚úÖ Education CRUD Operations

## üöÄ Getting Started

### **Prerequisites**
- Node.js 22
- Angular 20
- VS Code (recommended)

### **Installation Steps**

1. **Clone the Repository**
```bash
git clone <repository-url>
cd test-app
```

2. **Install Dependencies**
```bash
npm install
```

3. **Run Development Server**
```bash
npm run start
# or
ng serve
```

4. **Open Browser**
```
http://localhost:4200
```
